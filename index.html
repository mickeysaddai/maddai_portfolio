When not coding I enjoy playing the piano, wokring out and the outdoors

Ampe.js

Ampe.js is inspired by the West African game Ampe, a game I grew up playing outside as a kid in Ghama. The goal of the game is to eliminate all of the opponent's players.
This game utilizes Canvas, HTML5, and Vanilla Javascript DOM manipulation techniques.

ENDURANCE

Endurance is a clone of the web app version of the mobile app Runkeeper. The goal of the app is to log workout activities. The app utilizes Ruby on Rails, React-Redux, PostgresSQL, Javascript, and MapBox API.

ECOPOOL

Built with MERN stack, EcoRoute addresses the age-old question of, “Where should we meet?”. This webapp optimizes a list of address inputs by distance/time and suggests the most equidistant location to the users attending the event. This app utilizes MongoDB, Express, React-Redux, Javascript.
